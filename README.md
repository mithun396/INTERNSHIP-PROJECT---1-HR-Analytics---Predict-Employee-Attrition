# INTERNSHIP-PROJECT---1-HR-Analytics---Predict-Employee-Attrition

## 🚀 HR Attrition Risk & Predictive Analytics
Predicting employee resignation using Machine Learning and visualizing key drivers through Power BI.

## 📌 Project Overview
Employee attrition is a major challenge for organizations.
This project uses Data Analytics + Machine Learning + Business Intelligence to:
- Identify key factors driving employee resignation
- Predict high-risk employees
- Provide data-driven attrition prevention strategies

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA) on HR dataset
- Build a classification model to predict attrition
- Evaluate model using Accuracy, ROC-AUC & Confusion Matrix
- Create interactive Power BI dashboard
- Suggest actionable HR strategies

## 🛠 Tools & Technologies
- Python
- Pandas
- Seaborn
- Scikit-learn
- SHAP (Model Explainability)
- Power BI

## 📊 Power BI Dashboard Features
✔ Attrition by Department
✔ Attrition by Job Role
✔ Salary Band Analysis
✔ Promotion Impact
✔ Overtime vs Attrition
✔ Gender-wise Analysis
✔ KPI Cards (Attrition Rate, ROC-AUC)

## 🔍 Key Insights
- Employees working overtime show higher attrition probability.
- Low salary band employees are more likely to resign.
- Lack of promotion significantly increases resignation risk.
- Certain departments show comparatively higher attrition.
